Enables the Contact Form 7 plugin to create multi-step slider forms.

== Description ==

This is a simple WordPress plugin that creates same page multi-step slider forms. Copy and paste the form shortcodes into a multi-step form batch process. You create the slider with the added Contact Form 7 shortcodes .

== Installation ==

**Activate** the plugin through the 'Plugins' menu in WordPress.

- Your WordPress site must be running **PHP5** or better. This plugin will fail to activate if your site is running PHP4.

- Be sure that all of the **supported form plugins** are installed and activated.


== Frequently Asked Questions ==

= Where are the setting for the multi-form sliders? =

You will find the settings in `Contact->Multi-Step Slider` submenu in your WordPress admin panel.

= How can I make the multi-form slider responsive? =

The multi-form slider is designed to adapt on various device sizes.

= My multi-step slider is broken? =

Please contact me with a detailed example of the problem that you're experiencing.
